# 損益分岐点シミュレーター

FC（フランチャイズ）オーナー向けの収支シミュレーションツール。
**固定費・変動費・単価の3つを入力するだけ**で、損益分岐点・限界利益率・収支グラフ・損益表がリアルタイムに表示されます。

元々 Numbers で作っていた損益分岐点分析を、誰でも数字を入れ替えるだけで使える Web ツールにしたものです。

## 表示される指標

- **損益分岐点（黒字化に必要な販売数）** — 月にいくつ売れば黒字になるか
- **損益分岐点売上高** — 収支トントンになる売上
- **限界利益（1件あたり）** — 1件売るごとに残る利益（＝単価 − 変動費）
- **限界利益率** — 売上に対する利益の割合
- **1日あたり必要販売数** — 月25日換算の目安
- **売上と費用のグラフ** — 売上線が総費用線を上回る点が損益分岐点
- **損益シミュレーション表** — 販売数ごとの売上・費用・損益（赤字＝赤 / 黒字＝青）

## 使い方

`index.html` を開くだけ。ビルド不要の静的サイト（依存はグラフ描画の Chart.js のみ）。

- **固定費・変動費は項目ごとに入力可能** — 「家賃 ¥250,000」「人件費 ¥400,000」のように内訳で入れられ、合計が自動でシミュレーションに反映されます。「＋項目を追加」「×」で行の追加・削除ができます。
- **業種プリセット**（小規模サロン／中規模／大型店舗）で内訳ごと一括入力。
- **損益表の刻み**を 自動／1件／2件／5件… から選択可能。

## 計算式

```
限界利益（1件あたり） = 単価 − 1件あたり変動費
損益分岐点（販売数）   = 固定費 ÷ 限界利益
損益分岐点売上高       = 損益分岐点（販売数）× 単価
限界利益率           = 限界利益 ÷ 単価
```
