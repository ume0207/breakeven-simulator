# FC収支シミュレーター

FC（フランチャイズ）オーナー向けの収支シミュレーションツール。
**固定費・変動費・単価を入れるだけ**で、同じ数字を使って「損益分岐点」と「年間損益」を切り替えてシミュレーションできます。

元々 Numbers で作っていた損益分岐点分析を、誰でも数字を入れ替えるだけで使える Web ツールにしたものです。1ページ完結（上部のタブで2つのモードを切り替え）。

## 損益分岐点モード

- **損益分岐点（黒字化に必要な販売数）** — 月にいくつ売れば黒字になるか
- **損益分岐点売上高** — 収支トントンになる売上
- **限界利益（1件あたり）／限界利益率** — 1件売るごとに残る利益と、その割合
- **1日あたり必要販売数** — 月25日換算の目安
- **売上と費用のグラフ** — 売上線が総費用線を上回る点が損益分岐点
- **損益シミュレーション表** — 販売数ごとの損益（赤字＝赤 / 黒字＝青）。表示の刻み（自動／1件／2件…）を選択可能

## 年間シミュレーションモード

- **月ごとの販売数**を1〜12月それぞれ入力（一括設定／毎月一定・右肩上がり・季節変動パターン）
- **初期投資（開業費）** を入れると「何ヶ月目で回収できるか」を自動計算
- **年間売上高・年間総費用・年間損益・月平均損益・投資回収月** をKPI表示
- **月別グラフ**（売上・費用の棒＋月間損益の線）と**累計損益グラフ**（初期投資差引後・回収点マーカー）
- **年間損益表**（12ヶ月＋年間合計、累計損益）

## 使い方

`index.html` を開くだけ。ビルド不要の静的サイト（共通CSSは `styles.css`、依存はグラフ描画の Chart.js のみ）。
左で入れたコスト（固定費・変動費・単価）が両モードで共通して使われます。固定費・変動費は「項目名＋金額」の内訳で入力でき（「＋項目を追加」「×」で増減）、業種プリセットで一括入力もできます。

## 計算式

```
限界利益（1件あたり） = 単価 − 1件あたり変動費
損益分岐点（販売数）   = 固定費 ÷ 限界利益
損益分岐点売上高       = 損益分岐点（販売数）× 単価
限界利益率           = 限界利益 ÷ 単価
```
